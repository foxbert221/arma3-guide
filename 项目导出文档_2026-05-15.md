```json
  "类型": "多用途战斗机",
  "简介": "详细介绍...",  // 可转为output
  "首飞时间": "2011年",
  "研发单位": "沈阳飞机公司",
  "最大飞行速度": "2.0马赫",
  "最大航程": "3,000公里",
  // ... 其他技术参数
}
```

**SFT格式转换示例**：
```json
{
  "instruction": "请详细介绍歼-16战机的参数和特点",
  "input": "",
  "output": "歼-16是沈阳飞机公司为海军航空兵研发的新型多用途战机..."
}
```

**优势**：
- ✅ 立即可下载
- ✅ 5,800条结构化数据
- ✅ 包含中外装备（歼-20、99A、F-35、AK-47等）

## 🚨 重要提醒

### 数据集使用注意事项
1. **版权和许可**：确认数据集的使用许可（MIT、Apache 2.0、CC BY等）
2. **数据质量**：下载后务必人工检查样本，确保数据准确
3. **涉密信息**：军事数据需特别注意，不得使用涉密数据

### 技术细节
- **SFT格式标准**：`{"instruction": "...", "input": "...", "output": "..."}`
- **JSON编码**：确保使用UTF-8编码
- **数据平衡**：4个分类的样本量尽量均衡（每类2.5K左右）

### 如果链接再次失效
很多数据集链接会失效，建议：
1. 优先使用GitHub raw链接（更稳定）
2. 及时下载并本地备份
3. 记录数据的MD5值，确保完整性
